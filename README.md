# SQL Sales Data Analysis Project

## Project Objective
The objective of this project was to perform SQL-based analysis on a sales dataset to identify trends, customer behavior, revenue performance, referral effectiveness, and payment method distributions.

## Dataset Description
The dataset contains transactional sales records including:
- Products
- Revenue
- Referral sources
- Payment methods
- Coupon usage
- Customer IDs
- Order statuses

## Tools Used
- SQLite
- DB Browser for SQLite

  

## SQL Concepts Used
- SELECT
- WHERE
- GROUP BY
- ORDER BY
- COUNT()
- SUM()
- AVG()
- LIMIT

## Analysis Performed
- Total revenue analysis
- Revenue by product
- Orders by payment method
- Top customers by revenue
- Order status analysis
- Referral source analysis
- Coupon performance analysis
- Average order value analysis

## Key Findings
- Total revenue exceeded $1.26 million
- Instagram generated the highest referral revenue
- Credit Card transactions generated the highest payment revenue
- FREESHIP coupon generated the highest coupon revenue
- Several high-value customer transactions were identified
